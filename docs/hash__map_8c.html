production *  value 
)

Inserts a production rule into the internal hash table for the LL(1) parser.

Parameters
keyA unique index generated using a combination of a non-terminal and the terminal token that selects the rule.
valueA pointer to the production rule to be stored.
Note
The key must be uniquely derived from both the non-terminal producing the rule and the token determining its selection.
Warning
The caller is responsible for freeing the memory allocated for the production rule. The hash table will NOT free these values on exit.
