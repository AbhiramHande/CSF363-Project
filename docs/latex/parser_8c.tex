Computes the First set of a given non-\/terminal. 

The First set of a non-\/terminal $ X $ is computed as follows\+:
\begin{DoxyEnumerate}
\item If $ X \to Y_1Y_2 \dots Y_k $ is a production rule for some $ k \geq 1 $\+:
\begin{DoxyItemize}
\item If $ \alpha \neq \epsilon \in First(Y_1) $, then add $ \alpha $ to First(\+X).
\item If $ \epsilon \in First(Y_1) $, continue checking $ Y_2, Y_3, \dots, Y_n $.
\item Stop when a symbol $ Y_n $ is found where $ \epsilon \notin First(Y_n) $ or all $ Y_i $ are exhausted.
\item If $ \epsilon \in First(Y_i) $ for all $ i \leq n $, add $ \epsilon $ to First(\+X).
\end{DoxyItemize}
\item If $ X \to \epsilon $ is a production, then $ \epsilon $ is added to First(\+X).
\item If a non-\/terminal has multiple production rules, the First set is the {\bfseries{union}} of the First sets of all right-\/hand sides.
\end{DoxyEnumerate}

The LL(1) parsing process follows these steps\+:
\begin{DoxyEnumerate}
\item Push the start symbol of the grammar ({\ttfamily $<$program$>$}) onto an empty stack.
\item Set {\ttfamily tp} to the first symbol of {\ttfamily w}, the tokenized input string.
\item Use the parsing table {\ttfamily M} to decide parsing actions.
\item While (stack is not empty)
\begin{DoxyItemize}
\item If top of stack matches current token, pop stack and advance {\ttfamily tp}.
\item If top of the stack is a terminal but does not match {\ttfamily a}, {\bfseries{report a syntax error}}.
\item If {\ttfamily M}\textquotesingle{}s entry corresponding to the stack top and current token is {\ttfamily NULL}, {\bfseries{handle error recovery}} (panic-\/mode).
\item If $ M[X, a] = X \to Y_1Y_2 \cdots Y_k $\+:
\begin{DoxyItemize}
\item Output the production $ X \to Y_1Y_2 \cdots Y_k $.
\item Pop {\ttfamily X} from stack.
\item Push $ Y_k, Y_{k-1}, \cdots, Y_1 $ onto the stack in that order.
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Repeat until the stack is empty.
\end{DoxyEnumerate}
