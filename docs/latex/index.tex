This project is a custom-\/built compiler designed to process a toy programming language as a part of the course {\bfseries{CS F363 -\/ Compiler Construction}}. The compiler consists of two phases, including lexical analysis and syntax analysis. It takes source code as input, processes it through these stages, and outputs either a Parse tree or meaningful error messages.
